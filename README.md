# Rig — русский лендинг

Адаптивная копия визуала [rig.ai](https://rig.ai/) на русском языке.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080

## Структура

- `index.html` — разметка всех секций
- `styles.css` — стили и адаптив
- `main.js` — меню, форма waitlist, анимации при скролле
